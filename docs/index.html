Real World Usage

Here's how teams are using syft-installer in production:

# In your Docker container
FROM python:3.9
RUN pip install syft-installer

# In your Python script
import syft_installer as si
import os

# Get email from environment
email = os.environ.get('SYFTBOX_EMAIL')
if not email:
    raise ValueError("SYFTBOX_EMAIL environment variable required")

# Install and run
print(f"Installing SyftBox for {email}...")
si.install_and_run(email)

# Your SyftBox app code here
from syftbox.lib import SyftConfig
config = SyftConfig.load()
print(f"Running on datasite: {config.datasite_name}")
