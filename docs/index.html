The Complete API

Everything you need to install and manage SyftBox

Quick Start

# Install and run in one line
import syft_installer as si
si.run()  # Installs (if needed) and starts SyftBox

# Or separate install and run steps
si.install()        # Just install
si.install_and_run() # Explicit install + run

Status & Control

# Check status
si.status()         # Show detailed status
si.is_installed()   # Returns True/False
si.is_running()     # Returns True/False

# Control the daemon
si.stop()           # Stop SyftBox
si.run_if_stopped() # Start only if not running

Non-Interactive Mode

# For automation and scripts
session = si.install("email@example.com", interactive=False)
# Returns InstallerSession object

# Submit OTP programmatically
session.submit_otp("ABC123")

# Session provides installation state
print(session.email)
print(session.is_authenticated)

Under the Hood

Here's what syft-installer does behind the scenes:

# When you run si.run(), it:

1. Detects your platform (macOS/Linux, Intel/ARM)
2. Downloads the correct SyftBox binary
3. Sends OTP to your email for verification  
4. Creates necessary directories:
   - ~/SyftBox (data directory)
   - ~/.syftbox (config directory)
   - ~/.local/bin/syftbox (binary location)
5. Saves authentication tokens securely
6. Starts the SyftBox daemon in background
7. Monitors the process for successful startup

# All handled automatically with proper error handling!
