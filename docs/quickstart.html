üéØ Basic Usage

üöÄ One-Line Installation

The simplest way to install SyftBox:

import syft_installer as si

# One-liner installation - prompts for email and OTP
si.install()

This will:

  • ‚úÖ Prompt for your email address
  • ‚úÖ Send OTP to your email
  • ‚úÖ Prompt for OTP verification
  • ‚úÖ Download and install SyftBox
  • ‚úÖ Start the SyftBox client

üìß Pre-configured Email

Skip the email prompt by providing it upfront:

import syft_installer as si

# Pre-configure email to skip one prompt
installer = si.Installer(email="your.email@example.com")
installer.install()  # Only prompts for OTP

This approach is perfect when you know the email address in advance.

üîß Full Programmatic Control

Break down the installation into individual steps:

import syft_installer as si

# Create installer instance
installer = si.Installer(email="your.email@example.com")

# Step 1: Request OTP
installer.request_otp()
print("OTP sent to your email!")

# Step 2: Verify OTP (get from email)
otp = input("Enter OTP from email: ")
installer.verify_otp(otp)

# Step 3: Start the client
installer.start_client(background=True)

Perfect for automation or when you need custom logic between steps.

‚öôÔ∏è Advanced Configuration

Customize the installation with additional options:

Installation Modes

# Download only (no setup)
si.install(install_mode="download-only")

# Setup only (binary already downloaded)
si.install(install_mode="setup-only")

# Interactive (default)
si.install(install_mode="interactive")

App Installation

# Install specific SyftBox apps
si.install(install_apps="dataroom,whisper")

# Install multiple apps
si.install(
    install_apps="calculator,dataroom,whisper"
)

Debug Mode

# Enable debug output
si.install(debug=True)

# Or set environment variable
import os
os.environ["DEBUG"] = "true"
si.install()

Environment Variables

# Set before running Python
export INSTALL_MODE=download-only
export INSTALL_APPS=app1,app2
export DEBUG=true
export ARTIFACT_BASE_URL=custom.url

üîß Troubleshooting

Common issues and their solutions:

‚ùå Installation Fails

Problem: Package installation fails

Solution: Try upgrading pip and installing with verbose output:

pip install --upgrade pip
pip install -v syft-installer

üìß OTP Not Received

Problem: OTP email doesn't arrive

Solution: Check spam folder and verify email address:

installer = si.Installer(email="correct@email.com")
installer.request_otp()  # Try again

üö´ Permission Errors

Problem: Permission denied during installation

Solution: Use user installation or virtual environment:

pip install --user syft-installer
# or create virtual environment first

üêç Python Version Issues

Problem: Compatibility issues

Solution: Ensure Python 3.8+ is being used:

python3 --version
pip3 install syft-installer

Still having issues? Report a bug on GitHub or ask for help on OpenMined Slack.
