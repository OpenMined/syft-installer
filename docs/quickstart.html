2-Minute Quick Start

From zero to running SyftBox in your notebook.

1 Install syft-installer

pip install syft-installer

2 Install and Run SyftBox

One line to install and start SyftBox:

import syft_installer as si

# Install and run SyftBox
si.run()

# That's it! SyftBox is now running.
# You'll be prompted for your email and OTP code.

3 Check Status

See if SyftBox is running and get details:

# Check status
si.status()

# Check specific states
if si.is_running():
    print("SyftBox is running!")

# Stop when needed
si.stop()

4 Google Colab Magic

In Google Colab, syft-installer automatically detects your email:

# In Google Colab
import syft_installer as si

# Automatically uses your Google account email
si.run()

# Or specify a different email
si.run("custom@email.com")

5 Advanced Usage

For automation and CI/CD pipelines:

# Non-interactive installation
session = si.install("your@email.com", interactive=False)

# Submit OTP programmatically
session.submit_otp("ABC123")

# Check installation details
print(f"Email: {session.email}")
print(f"Server: {session.server}")
print(f"Authenticated: {session.is_authenticated}")

# Clean uninstall when needed
si.uninstall()  # Removes everything
