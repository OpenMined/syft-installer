Installation

Install syft-installer to get SyftBox up and running with minimal effort.

Quick Install

Install syft-installer from PyPI with pip:

pip install syft-installer
✅ That's it! Now you can install and run SyftBox with just si.run(). Jump to the Quick Start guide to begin.

What syft-installer Does

🎯 One Command Setup

Install and run SyftBox with a single Python command:

import syft_installer as si si.run() # That's it!

This handles:

  • Platform detection (macOS/Linux)
  • Binary download for your architecture
  • Email-based authentication
  • Directory setup and configuration
  • Daemon process management

🚀 Perfect for Notebooks

Optimized for Jupyter and Google Colab environments:

# Google Colab auto-detects your email si.run() # Uses your Google account

Special features:

  • Rich terminal output formatting
  • Interactive OTP prompts
  • Progress bars for downloads
  • Clean error messages
  • Automatic environment detection

Usage Examples

Common scenarios for using syft-installer:

Interactive Installation (Default)
import syft_installer as si si.run("your@email.com") # Prompts for OTP interactively

Best for manual installations where you can enter the OTP when prompted. The installer will wait for you to check your email and enter the code.

Non-Interactive Installation
session = si.install("your@email.com", interactive=False) # Later, when you receive the OTP: session.submit_otp("ABC123")

Perfect for automation, CI/CD pipelines, or when you need to handle OTP submission programmatically.

Process Management
si.status() # Check if running si.stop() # Stop the daemon si.run_if_stopped() # Start only if not running si.uninstall() # Complete removal

Full control over the SyftBox daemon lifecycle. Useful for managing SyftBox in production environments.

System Requirements

📋 Minimum Requirements

  • Python: 3.9 or higher
  • Operating System: macOS or Linux
  • Architecture: x86_64 or ARM64
  • Memory: 100MB RAM
  • Disk: 200MB storage

🎯 Recommended Setup

  • Python: 3.10+ for best performance
  • Package Manager: pip 21+ or uv
  • Environment: Jupyter or Google Colab
  • Network: Internet for OTP verification

Want to contribute to syft-installer or work with the source code? Follow these steps:

⚡ Quick Setup

Get up and running in under 2 minutes:

  1. Clone the repository
  2. Install dependencies
  3. Run tests
  4. Start coding!
Step 1: Clone the Repository
git clone https://github.com/OpenMined/syft-installer.git cd syft-installer
Step 2: Create Virtual Environment (Recommended)
python -m venv venv source venv/bin/activate # On Windows: venv\Scripts\activate
Step 3: Install in Development Mode
pip install -e ".[dev]"

This installs syft-installer in editable mode with all development dependencies including:

  • pytest: Testing framework
  • pytest-asyncio: Async test support
  • pytest-cov: Code coverage reporting
  • ruff: Fast Python linter
  • mypy: Type checking
Step 4: Verify Installation
pytest # Run all tests pytest -v # Run with verbose output pytest tests/test_permissions.py # Run specific test file pytest tests/test_notebooks.py # Run notebook tests
💡 Pro Tip: The test suite includes automated testing of tutorial notebooks to ensure they stay up-to-date and functional. Make sure you have the required dependencies installed (nbformat, nbconvert, and ipykernel) before running notebook tests.
💡 Pro Tip: We recommend using uv for faster dependency management:
pip install uv uv pip install -e ".[dev,server,display]" # 10x faster than pip!

Verify Installation

Confirm that syft-installer is installed correctly:

python -c "import syft_installer; print(f'syft-installer {syft_installer.__version__} installed successfully!')"

You should see output like:

syft-installer 0.8.24 installed successfully!

Test that the API is available:

import syft_installer as si # Check available functions print("Available functions:") for func in ['run', 'install', 'status', 'stop', 'uninstall']: print(f" - si.{func}()") # Check installation status if si.is_installed(): print(" ✅ SyftBox is already installed!") print(f"Running: {si.is_running()}") else: print(" 📦 SyftBox not installed yet. Run si.run() to install!")

Troubleshooting

Platform Not Supported

Problem: Platform Windows is not supported

Solution: syft-installer currently supports macOS and Linux only. Windows support is coming soon.

Import Error

Problem: ModuleNotFoundError: No module named 'syft_installer'

Solution: Ensure you're using the correct Python environment. If using virtual environments, activate it first.

OTP Not Received

Problem: Email with OTP code doesn't arrive

Solution: Check spam folder. The email comes from SyftBox authentication service. If still not received, try again with a different email.

Daemon Won't Start

Problem: Failed to start SyftBox daemon

Solution: Check if another instance is running with si.status(). Stop it with si.stop() before trying again.
